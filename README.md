# calculator-gemini

Modern, responsive, and functional calculator application built with Vanilla TypeScript and Vite.

## Features
- Basic arithmetic operations (+, -, *, /).
- High information density, low noise (Quant/Minimalist style).
- Keyboard support.
- Automated deployment to GitHub Pages.

## Live Demo
🚀 **[View Live Application](https://Ramasanjaya22.github.io/calculator-gemini/)**

## Tech Stack
- Frontend: Vanilla TypeScript
- Build Tool: Vite
- Styling: Vanilla CSS
- Deployment: GitHub Actions

## Development
```bash
npm install
npm run dev
```
